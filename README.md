# Purevibe

A modern, highly interactive, and premium mobile-first landing page for "Purevibe" jelly nutritional supplements. Built with HTML, CSS, vanilla JavaScript, and [GSAP (GreenSock)](https://greensock.com/gsap/) for smooth scroll-driven animations.

**[🌐 View Live Site on GitHub Pages](https://balee-afk.github.io/purevibe-threejs/)**

## ✨ Features
- **GSAP ScrollTrigger Animations**: Dynamic product fan-out animations, smooth parallax interactions, and cinematic content reveals seamlessly tied to the user's scroll.
- **Dynamic Color Palettes**: Background colors transition smoothly according to the active product variant (e.g., green for Vibrant Green, pink for Collagen & Berry).
- **Responsive Layout**: Designed specifically to look stunning on mobile devices while maintaining an elegant presentation on desktop screens.
- **Interactive Navigation**: 
  - Off-canvas sidebar menu with "frosted glass" (glassmorphism) effects.
  - Interactive pagination dots for rapid slide navigation.
  - "Back to Top" scrolling functionality.
- **Premium Aesthetics**: High-end typography, modern dark mode design, and sleek UI spacing.

## 🚀 Running Locally

To run this project locally, you can use PHP's built-in development server, or any standard HTTP server (like VSCode Live Server, Python `http.server`, etc.).

```bash
# Clone the repository
git clone https://github.com/balee-afk/purevibe-threejs.git

# Navigate into the directory
cd purevibe-threejs

# Start a local PHP server (requires PHP installed)
php -S localhost:8000
```
Then visit `http://localhost:8000` in your web browser.
